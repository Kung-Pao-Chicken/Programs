1.安装编译需要的环境：
[plain] view plaincopyprint?
  1. sudo apt-get install python g++ make checkinstall  

2.从官网上获取源代码并解压：

[plain] view plaincopyprint?
  1. mkdir ~/src  
  2. cd ~/src  
  3. wget -N http://nodejs.org/dist/node-latest.tar.gz  
  4. tar xzvf node-latest.tar.gz && cd node-v*  

3. 配置编译源代码

[plain] view plaincopyprint?
  1. ./configure --without-snapshot //注意这个地方一定要加入这个参数，否则之后在运行时有可能会出现段错  
  2.                               //误的提示。  
  3. sudo checkinstall  

4.安装和卸载

[plain] view plaincopyprint?
  1. sudo dpkg -i node_*  
  2. sudo dpkg -r node  

5.检测是否正常工作

在TX终端中输入node，再输入：

[plain] view plaincopyprint?
  1. Console.log("DoubleVoid'sDemo,转载请注明~");
