最近新入手了一块CubieTruck的板子，激动啊~但是原装Android实在是让我蛋疼不少...果断官网下载Debian，我使用的是有可视化界面的，直接写入固件中，而不是TF卡的版本，下载地址：debian-chird-desktop-nand.img.gz，我用的学校实验室的网，速度还可以，具体怎么烧录我就不多说了，网上教程满天飞...安装完成后第一件事就是让无线网能正常使用：

1.加载模块:
[plain] view plaincopyprint?
  1. sudo modprobe bcmdhd  
2.安装wicd 无线网管理器:
[plain] view plaincopyprint?
  1. sudo apt-get install wicd  
有 人说为啥不用自带的network-manager呢？我只想说，自带的那个nm实在是太烂了，如果一定要使用的话，不仅要改它的配置文件，再改/etc /network/interfaces，而且还十分不稳定~所以我果断放弃之。之后wicd和network-manager并不冲突，非处女座可无视 之...

3.连接网络:
直接右下角→互联网→Wicd 
Network Manager→首选项→常规设置→无线网络，在里面填入wlan1，这个地方因板子而异，我的是wlan1，查看方法是在LX终端输入ifconfig，其中带wlan的就是你的网线网卡端口。填到里面就行了。

4.OK，大功告成！享受上网吧~
